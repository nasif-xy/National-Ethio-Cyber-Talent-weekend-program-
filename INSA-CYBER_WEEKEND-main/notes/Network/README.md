##Networking

Networking = connecting devices to share data (computers, phones, servers)
📡 Types (by Geographical Coverage)
LAN (Local Area Network)
→ Small area (home, school)
WAN (Wide Area Network / Internet)
→ Large area (countries, world)
WLAN (Wireless LAN)
→ LAN but wireless (Wi-Fi)
PAN (Personal Area Network)
→ Very small (Bluetooth, phone)
MAN (Metropolitan Area Network)
→ City-level network
🌍 IP Address
Unique address for each device on a network
Example: 192.168.1.1
🔢 IP Classes (IPv4)
🅰️ Class A
Range: 1.0.0.0 – 126.255.255.255
Default mask: 255.0.0.0
Large networks
🅱️ Class B
Range: 128.0.0.0 – 191.255.255.255
Mask: 255.255.0.0
Medium networks
🅲 Class C
Range: 192.0.0.0 – 223.255.255.255
Mask: 255.255.255.0
Small networks
🅳 Class D
Range: 224.0.0.0 – 239.255.255.255
Used for multicast (group communication)
🅴 Class E
Range: 240.0.0.0 – 255.255.255.255
Reserved (experimental)
🧠 Permutation vs Combination (for IP)
👉 In IP addressing, we use combinations, not permutations
Combination → order doesn’t matter (IP ranges)
Permutation → order matters (not used here)
