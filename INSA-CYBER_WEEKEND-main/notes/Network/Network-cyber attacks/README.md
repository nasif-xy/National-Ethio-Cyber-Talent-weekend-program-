##Network-related cyber attacks

Network-related cyber attacks target different parts of a network (often mapped to OSI layers). 

 Common Network Cyber Attacks

🧨 1. DoS / DDoS (Denial of Service)
Floods a server/network with traffic
Makes services unavailable
Types:
SYN Flood (Transport layer)
HTTP Flood (Application layer)
👉 Example: A website crashes because of massive fake traffic

🎭 2. Man-in-the-Middle (MITM)
Attacker secretly intercepts communication
Can read or modify data
👉 Example: You connect to public Wi-Fi → attacker captures your login info

🕵️ 3. Packet Sniffing
Capturing network traffic
Used to steal passwords, cookies
👉 Tools: Wireshark, tcpdump
👉 Works best on unencrypted networks

🎣 4. Phishing (Network-assisted)
Fake websites/emails trick users
Often combined with DNS or MITM attacks

🧬 5. DNS Spoofing (DNS Poisoning)
Redirects users to fake websites
Changes IP address mapping
👉 Example: Typing google.com → goes to attacker’s site

🛜 6. ARP Spoofing
Attacker links their MAC address to victim’s IP
Used to perform MITM inside local networks

🔁 7. Replay Attack
Attacker captures data and resends it later
Tricks system into thinking it's valid

🧱 8. Session Hijacking
Takes over an active session
Steals session cookies
👉 Example: Attacker logs into your account without password

📡 9. Wi-Fi Attacks
Evil Twin (fake Wi-Fi)
Deauthentication attack (kicks users off network)
